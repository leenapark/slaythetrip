# slaythetrip

## 개요
- Python django를 이용한 여행 관리 웹 프로젝트

#### 목적
- 여행할 때 필요한 부분을 위해 여러 사이트를 이용해야 하는 불편함 해소

#### 개발 기간
- 2023.06.15~2023.06.29

#### 플랫폼
- Web

#### 개발 인원
- 4명

#### 담당 역할
- 서비스 기획 및 방향성 설정 (기여도 90%)
- 프론트엔드 (기여도 70%)
- 웹 사이트 기술 지원 (기여도 80%)
- 호텔 페이지 (기여도 20%)
- 맛집 페이지 (기여도 50%)
- 날씨 페이지 (기여도 60%)
- Merge & Debug

### 개발 환경
- 언어
  - Python 3.9
  - HTML/CSS
  - JavaScript
- 프레임워크
  - Django
- ML
  - LogisticRegression 
- DB
  - MySQL
- IDE
  - VS Code
  - DBeaver
- API, 라이브러리
  - Jquery
    - ajax
  - fetch
  - numpy
  - Pandas
  - joblib
